原生go top 回到頂端的方法

其實在原生中就有一種gotop的方法，根本就不需要去使用JQuery套件。

除了使用錨點外，還可以使用scrollIntoView()這個方法，這個方法除了擁有錨點的跳轉外，還有平滑移動的動畫效果

那我們就直接先來看code吧!

    document.querySelector('#step3').scrollIntoView({
        block:'center', //垂直卷軸的位置
        behavior:'smooth',//轉移方式
        inline:'center'水平卷軸的位置
    });

首先scrollIntoView地使用邏輯是這樣的:

我們指定的dom會將會是畫面要移動到的位置。其次裡面的屬性分別是垂直卷軸的位置、轉移方式、水平卷軸的位置。

block(垂直卷軸位置):

這個屬性決定了垂直捲軸的位置，接受 start、center、end 和 nearest 四種。預設為 start

用途為移動時對其卷軸的位置。其中 start 代表對齊垂直捲軸的頂部，end 對齊垂直底部，center 自然就是對齊中心點，而 nearest 則是依據元素當下位置來判斷最近的位置來對齊

inline(水平卷軸位置):

這個屬性則是決定了水平捲軸的位置。接受 start、center、end 和 nearest 四種。預設為 nearest

start代表水平卷軸的左側。end代表水平右側、center 對齊中心點。至於nearest與垂直相同。

behavior(轉移方式):

這個屬性共有兩個值，決定轉移方式為跳轉or動畫滑動

auto為跳。smooth則是平滑移動。預設則是 auto 跳的

go top

因此若是要使用gotop回到頂端，就將目標位置對向最頂端的dom即可。
