HTML基本架構介紹

廢話不多說我們直接看code及註解介紹

    <!DOCTYPE html> //宣告這是HTML網頁
    <html lang="en">//宣告頁面主體語言
    <head>
        <meta charset="UTF-8">//編碼格式
        <meta http-equiv="X-UA-Compatible" content="IE=edge">//兼容IE
        <meta name="viewport" content="width=device-width, initial-scale=1.0">//自適應版面大小
        <title>Document</title>//網頁標題
        //最基礎的架構↑
    
        <link rel="shortcut icon" href="favicon.ico"> //標題圖示
        <meta name='description' content='網站描述文字' /> //於搜尋引擎看到的網站介紹
    
        <meta property="og:title" content="FB/line等軟體上的標題">
        <meta property="og:description" content="FB/line等軟體上的描述">
        <meta property="og:type" content="website"> //宣告這是網頁
        <meta property="og:url" content="FB/line等軟體上的網址">
        <meta property="og:image" content="FB/line等軟體上的圖片">
        <meta name="keywords" content="Holdkibor, holdkibor, JQuery, Javascript, HTML, JS, CSS">
        //給予搜尋引擎的關鍵字 使用者搜尋關鍵字就可以找到你的網頁
    
        <link href="圖片路徑" rel="apple-touch-icon" /> 放在手機捷徑的圖示
    
        <link rel="stylesheet" href="CSS檔案路徑"> //導入外部CSS樣式
    </head>
    <body>
        顯現的內容
        <script type="text/javascript" src="JS檔案路徑"></script> 導入外部JavaScript
    </body>
    </html>
