使用click事件完成dblclick事件

我自己在開發時曾經遇到過想使用dblclick事件卻因為與插件使用的API衝突的困擾發生。

為了使自己能夠使用api完成自己想要的功能，所以就自己就花了點時間研究出這個變相的方法來


    let select_num = 0;//首先在全域先定義個0來當作判斷的工具。

    function dblclick(){
        select_num++;//觸發事件後讓select_num +1
        setTimeout(() => {/200毫秒後select_num歸零
            select_num = 0;
        }, 200);
        if(select_num == 2){//若在兩百毫秒內再次觸發事件，那個select_num就會因為++而從1變成2
            //當select_num說明了在兩百毫秒內連續觸發了兩次事件，若用在click上就等於滑鼠雙擊了兩次
            //這正是dblclick事件，在if這邊就可以寫上你雙擊後所要觸發的功能。
            select_num = 0;
        }
    }

以上就是今天所分享的使用click事件來完成dblclick事件，若有疑問可以與我聯絡。
